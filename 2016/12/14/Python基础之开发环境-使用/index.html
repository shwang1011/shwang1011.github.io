常用命令介绍

1. install

注意: 安装时pyenv是从Python官网下载，速度会比较慢。我们可以将下载好的Python源码包(如：Python-3.5.2.tar.xz)放置于~/.pyenv/cache/下(如果没有cache目录，请自行创建)，再进行install安装，这就省去了从Python官网下载指定Python版本的过程。
