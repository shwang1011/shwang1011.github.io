1.2.3 逻辑运算符

参与运算的成员只能是布尔类型，或者
逻辑运算符:
and 与，双目运算
or 或，双目运算
not 非，单目运算
and：

True and True = True
True and False = False
False and True = False
False and False = False

or：

True or True = True
True or False = True
False or True = True
False or False = False

not：

not True = False
not False = True
